ServerManager!


If you need a moderation bot, you should get @ServerManager#9610. It can ban, unban, clear and kick an is easy-to-use.


Command prefix: ^


Commands:

  • clear -- Clears a certain amount of messages (Permissions: (User: Manage Messages), (Bot: Manage Messages))
    ban -- Bans a user (Permissions: (User: Ban Members), (Bot: Ban Members))
    unban -- Unbans a user (Permissions: (User: Ban Members), (Bot: Ban Members))
    kick -- Kicks a user (Permissions: (User: Kick Members), (Bot: Kick Members))
    guide -- Gives a link for the official guide (Permissions:(User: Send Messages), (Bot: Send Messages))
    invite -- Gives an invite link (Permissions: (User: Send Messages), (Bot: Send Messages))
